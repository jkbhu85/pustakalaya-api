Common Responses

These are common responses returned by the API. Response type is application/json. See HTTP response codes. See error codes.

The format of the response is as following:

{
    "responseCode": <xx>
    "message": "",
    "errors": {},
    "data": <primitive_type> | {} | []
}
A response may or may not have all the fields.
Requester is not authenticated.
401
No response message.
Requester is not authorized.
403
No response message.
Any other error.
500
No response message.
Data is invalid.
422 
{
    "responseCode": 61,
    "message": "ERROR_INVALID_FIELDS",
    "errors":  {
        "<INVALID_FIELD_NAME>": <3x>
    }
}

Send appropriate error codes.
