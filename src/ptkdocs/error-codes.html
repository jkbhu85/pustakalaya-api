Error Codes

These codes can be sent with resposne messages from API endpoint to client. These codes are meant to be sent with HTTP response code 200.

Successful responses

10

Operation is successful

Security related responses

21

Invalid credentials (logging in)

22

Account locked

23

Access revoked

24

Password didn't match (other than logging in)

25

Security answer didn't match

Validation related

31

Value too large

Date type Interpretation
number value of numer
string length of string
date forward then the system was expecting
file Size of the file
32

Value too small or empty (for empty file, use code 37)

Date type Interpretation
number value of numer
string length of string
date backward then the system was expecting
file Size of the file
33

Invalid format

34

Unsupported file extension

35

Value already exists

36

Resource has expired

37

Empty file

System related

41

System unavailable

42

Mail could not be sent

Unsuccessful responses

51

Operation unsuccessful

52

Unknown error (The system did not anticipate this error.)

53

Limit exceeded

54

Resource does not exist
